# Unity_bumpkins

Unity remake van **Beasts & Bumpkins** (1996, Bullfrog).  
Doel: minimale speelbare core loop met productie, shops, bouwen en een happiness meter.

## Docs

- [DESIGN.md](DESIGN.md) — core loops, data model, entities, state machine
- [PLAN.md](PLAN.md) — jam plan, taken per stap, DoD, einddoel

## Assets

PNG sprites geëxtraheerd uit het originele spel via BBTools en opgeschaald met ESRGAN.  
Locatie: `Assets/`

## Quickstart

1. Open project in Unity
2. Bekijk [PLAN.md](PLAN.md) voor de volgende stap
